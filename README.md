• Personal Finance Manager – Android Application
Personal Finance Manager is a modern Android application developed to help users efficiently manage their financial activities. The application replaces traditional paper-based tracking and fragmented tools with a secure, automated, and user-friendly digital solution.
This project focuses on financial tracking, budgeting, goal monitoring, financial calculators, secure document storage, and real-time financial news integration.

• Key Features
-> Income and expense tracking
-> Graphical representation of financial data
-> Monthly and category-based budgeting
-> EMI,SIP and Tax calculator
-> Real-time financial news updates

• Tech Stack
Language: Kotlin
Database: Firebase
APIs: Financial News API 
IDE: Android Studio

• How to Run
1. Clone the Repository
git clone https://github.com/Sibasish45/minorproject1.git

Open Android Studio

Allow all dependencies to download and sync successfully.
4. Run the Application
